O builder não ira funcionar por padrão, você precisa de ter o
pyinstaller no python,

para isso:
1 - abra o cmd
2 - digite: pip install pyinstaller
pronto

2 - coloque em uma pasta qualquer
3 - quando for usar coloque o caminho para a pasta
